# crypto-tracker

This project is a web-based application that displays cryptocurrency data using a public API. The application allows users to explore different cryptocurrencies and view important financial details such as price, market cap, etc.

# API Used:
CoinGecko API
Endpoint:
https://api.coingecko.com/api/v3/coins/markets?vs_currency=inr

# Features:
 -Search cryptocurrencies by name or symbol
 -Filter top cryptocurrencies based on market cap rank
 -Sort cryptocurrencies by price and market cap
Bonus Features (Planned)
 -Dark mode toggle
 -Loading indicator while fetching data
 
# Technologies Used:
HTML
CSS
JavaScript (Fetch API, Array Higher Order Functions)

# Objective:
The objective of this project is to build a responsive web application using JavaScript, integrate a public API, and implement dynamic features like search, filter, and sorting using array higher-order functions.
